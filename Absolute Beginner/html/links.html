Useful Links
In the section below, the author has included useful Python programming-related links to aid you in your learning more about the subjects. Your computer must be connected to the World Wide Web for these links to work.

 

Python's Home:

This is where you will find the latest release of Python along with news, information, and other resources.

http://www.python.org

The Python Software Foundation:

The PSF is a non-profit membership organization devoted to advancing open source technology related to the Python programming language.

http://www.python.org/psf

Python User Groups:

A list of groups that meet in person to discuss all things Python.

http://www.python.org/UserGroups.html

Python City:

A comprehensive list of Python resources, including book reviews and a series of tutorials.

http://www.awaretek.com/plf.html

Python Links Collection:

A collection of additional links where you can find more information about Python programming.

http://www.cetus-links.org/oo_python.html

Python Tutor Mailing List:

The list is for people who want to ask questions regarding how to learn computer programming with the Python language. 

http://mail.python.org/mailman/listinfo/tutor

Pygame:

Hosts Pygame, a set of modules for writing games.

http://www.pygame.org/

Vaults of Parnassus:

Lists hundreds of Python programs and packages organized into more than a dozen categories.

http://www.vex.net/parnassus/

Starship Python:

A hosted site for the Python community.

http://starship.python.net/

VPython:

Home of an easy-to-use Python module for real-time 3D output.

http://vpython.org/

py2exe:

Offers a way to convert Python scripts to stand-alone Windows executables.

http://starship.python.net/crew/theller/py2exe/
