Python:

Website: http://www.python.org/

The complete Python programming language. Python is the interpreted, interactive, object-oriented programming language used in this book.

The Python implementation is copyrighted but freely usable and distributable, even for commercial use. (For more information about copyright, follow this link, which requires a live Internet connection: http://www.python.org/doc/Copyright.html)

  • Install Python 2.2.3 from this CD-ROM (Open from current location if prompted)
  • Read Me

Python is open source.

The Gimp for Windows:

Website: http://www.gimp.org/~tml/gimp/win32/

The Gimp is the GNU Image Manipulation Program. It is a freely distributed piece of software suitable for such tasks as photo retouching, image composition and image authoring.

  • Open the GTK+ 1.3 Runtime Environment Archive on this CD-ROM (This package is required by The Gimp.)
  • Open The Gimp Archive on this CD-ROM

The Gimp for Windows is GNU (http://www.gnu.org) licensed.
